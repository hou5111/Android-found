# Android通讯录APP

基于Android原生开发的通讯录应用程序，实现联系人管理功能。

## 功能特性
- 联系人增删改查
- 联系人分组显示
- 快速拨号
- 搜索功能
- 数据持久化存储

## 技术栈
- Java
- Android SDK
- SQLite数据库

## 快速开始
1. 克隆项目：`git clone https://github.com/hou5111/Android-found.git`
2. 使用 Android Studio 打开
3. 连接设备或模拟器运行
